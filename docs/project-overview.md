# Nuestra Gran Aventura

A premium web application created as a retirement gift.

Purpose:

Allow my parents to explore and dream about future cruises and destinations.

Main goal:

Celebrate retirement and create unforgettable memories.

The trip does not have a fixed date.

The experience should feel:

- Emotional
- Elegant
- Relaxed
- Premium
- Inspiring

Inspirations:

- Apple
- Airbnb
- Four Seasons
- Celebrity Cruises

Audience:

Two retired adults.

First cruise experience.
